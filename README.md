# Walker's Blog

这是我的个人博客，使用 Hexo + NexT 主题搭建，托管在 GitHub Pages。

## 访问地址

https://walkerjohnsonofficial.github.io

## 技术栈

- [Hexo](https://hexo.io/) - 快速、简洁且高效的博客框架
- [NexT](https://theme-next.org/) - 优雅而强大的 Hexo 主题
- [GitHub Pages](https://pages.github.com/) - 免费的静态网站托管服务

## 本地运行

```bash
npm install
npm run server
```

访问 http://localhost:4000 查看效果。

## 部署

```bash
npm run build
npm run deploy
```
